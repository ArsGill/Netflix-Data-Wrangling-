# Netflix Data Wrangling Project

###  Overview
This project focuses on **data cleaning, preprocessing, and exploratory data analysis (EDA)** on the Netflix dataset.  
The goal was to prepare the data for deeper insights and future modeling tasks using Python data science libraries.

---

###  Objectives
- Load and inspect raw Netflix data  
- Handle **missing values** and inconsistent data types  
- Extract and engineer new features (e.g., release year, duration type)  
- Perform **data wrangling** to create a structured and clean dataset  
- Visualize insights using **Matplotlib**
